要解决此问题，我们可以使用以下步骤：

1. 使用LEFT JOIN连接两个表。
2. 以部门名称和部门ID为基准分组。
3. 使用COUNT函数统计每个部门的学生数量。
4. 使用IFNULL函数确保即使某些部门没有学生也可以返回0作为计数。

考虑到上述步骤，查询如下：

```sql
SELECT 
    d.部门id,
    d.部门名称,
    IFNULL(COUNT(s.学生id), 0) AS 学生人数
FROM 
    部门表 d
LEFT JOIN 
    学生表 s ON d.部门id = s.部门id
GROUP BY 
    d.部门id, d.部门名称;
```

在这里，“部门表”和“学生表”分别代表部门和学生的表名。这个查询将返回每个部门的ID、名称和学生人数。
